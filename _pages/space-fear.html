*[WIP] Space Fear

A text-based horror adventure set on a spacestation. Click the red power button to enter standby mode, and again to power up the terminal. Follow the on-screen instructions to start. Type "help" for a list of commands.

Built using the Canvas Rendering Context Web API. Not mobile friendly See the GitHub page for more details.
